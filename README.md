# myProject

## System dependencies
* Ruby `2.7.2p137`
* Rails `6.0.6.1`
* Bundler `2.4.22`

## Comands

### Configuration

* install gem
```bash 
$ bundler install
```
* server
```bash
$ rails s
```
* console
```bash
$ rails c
```

### Database creation
* initialization
```bash 
$ sudo service postgresql start
```
* create
```bash
$ rails db:create
$ rails db:migrate
```

### How to run the test
```bash
bundle exec rspec
```
